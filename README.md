# Mathlab

Mathlab is a math-tutor workspace for building reusable lessons, practice tasks, and future assessment tooling.

The immediate goal is to create notebook-based classes that build from fundamentals toward signal processing. The long-term goal is to turn the material into a learning webapp where students read lessons, complete tasks, receive assessment, and get agent-guided corrections.

## Course Tracks

- Linear algebra: vectors, dot products, projections, bases, eigenvectors
- Calculus: functions, limits, derivatives, optimization
- Integrals: accumulation, area, antiderivatives, numerical integration
- Matrix theory: matrix-vector products, transformations, systems, eigentheory
- Geometry: coordinate geometry, vectors, transformations, curves
- Signal processing bridge: signals as vectors, similarity, orthogonality, Fourier intuition

## Repository Layout

- `docs/roadmap.md`: course roadmap and class sequencing
- `docs/future_webapp.md`: future webapp idea and product notes
- `docs/notebook_interactivity_plan.md`: interactive notebook design and implementation plan
- `notebooks/`: lesson notebooks organized by track
- `notebooks/_helpers/`: reusable notebook exercise helpers

## How We Will Use This

Each notebook is intended to work like a class:

1. Learn the core concept.
2. Walk through a worked example.
3. Try practice tasks.
4. Check answers and common mistakes.
5. Connect the idea to signal processing when useful.
